# AGENTS.md

## Propósito del repositorio
Este repositorio es un proyecto académico guiado para la entrega final del curso de DevOps.

## Reglas del proyecto
- El desarrollo se divide en etapas.
- No se implementa Docker, Terraform, Kubernetes ni GitHub Actions en la primera etapa.
- La primera etapa es solo una API Express mínima con pruebas.
- Priorizar simplicidad, costos bajos, seguridad básica y claridad para explicar.
- No hacer commits ni pushes sin autorización explícita.

## Instrucciones para el asistente
- Esperar la aprobación antes de avanzar entre etapas.
- Confirmar cada cambio significativo con el estudiante.
- Mantener la solución simple y explicable.
- Documentar claramente cada paso.

## Límite actual
- Etapa 1: app Express mínima con:
  - `GET /`
  - `GET /health`
  - `POST /echo`
  - tests automatizados

## Nota
Este archivo debe mantenerse en la raíz del proyecto y reflejar el enfoque guiado y gradual del trabajo.
